你不知道的Javascript

发表于 2016-11-15   |   分类于 Javascript   |  

一、Javascript简介

1、JavaScript由三部分组成：
ECMAScript，由ECMA-262定义，提供核心语言功能；
文档对象模型（DOM），提供访问和操作网页内容的方法和接口；
浏览器对象模型（BOM），提供与浏览器交互的方法和接口。

二、基本语法

3.1.1、Javascript区分大小写。
3.1.2、标识符是指变量、函数、属性的名字，或者函数的参数，而且标识符第一个字符必须是字母或者下划线__或者美元符号$，其他的字符可以是数字和字母。一般javascript采用的是驼峰命名法。不能把关键字、保留字当作标识符。其实标识符就是一些名字而已。
3.1.3、严格模式，在第五版中引入严格模式，如果要在整个脚本中加入严格模式，只要在脚本最上面加上”use strict”，如果只想在函数中启用严格模式，只需要在函数中加上”use strict”.
3.1.4、语句：一般语句结束最好加上分号;把多条语句用花括号包起来。

3.2、关键字和保留字
关键字就是表示种控制语句的开始或者结束或执行特定的操作的单词。保留字是js保留起来的关键字，这些字都不能用作变量的名字。

3.3、变量
变量是指储存数据类型的，一般声明变量和初始化一起，var b=10;这个意思是声明一个变量b并且使她的值等于10，而且使用var声明的变量是局部变量。如果在函数这var一个变量，那么这个函数执行完毕，这个变量就会销毁。

3.4、数据类型
在第五版中规定了6中数据类型：Number，String，Boolean，Null，Undefined，Object，前面五个也叫基本类型，最后的是复杂数据类型，注意说数据类型的时候，他们的头字母都是大写的。用typeof操作符来操作返回的字符串头字母都是小写的。
3.4.1、Undefined数据类型
typeof Undefined //“undefined”,只有一个值，即undefined，一般这个没啥用，当程序检测出来这个就表示程序出问题了。
3.4.2、Null数据类型
typeof Null //“null”，返回的是一个字符串null，值也只有一个：null，这个一般是变量如果以后用于保存对象，那么先初始化变量为null
3.4.3、Boolean数据类型
typeof 返回的是”boolean”,他有两个值，ture,false。但是他们和数字1，0不一定是相等的，因为是不同的数字类型，如果通过隐式转换可能是相等的。而且ture和false是区分大小写的。任何一个数可以通过Boolean()函数来转换成布尔值。转换的规则是任何非空字符串转换成ture，空字符串转换成false。任何非0的数字转换成true，0和NaN转换成false。任何对象转换成ture，null转换成false。undefined转换成false。
3.4.4、Number数据类型
（1）最基本的是十进制，就是一般的整数，比如55，当然还可以用八进制来表示，八进制第一位是必须是0，然后是八进制的数字（0-7），如果超过则解析成十进制，比如078，因为8超过了，所以js解析成78，但是在严格模式下八进制是无效的，会报错。十六进制是以0x开头的，后面跟任何十六进制数（0-9及 A-F），在进行数字运算的时候，最后都要换算成十进制。
（2）八进制转换成十进制 比如八进制12=1*8 ，从左到右降序。
将十进制数115转化为八进制数 ，除以8。8| 115…… 3 ，8| 14 …… 6 ，8| 1 …… 1 ，结果：（115）10 = （163）8
16进制的20表示成10进制就是：2×16¹+0×16º=32，也是从左到右降序。
采余数定理分解，例如将487710转成十六进制：
487710÷16=30481….14(E)
30481÷16=1905….1
1905÷16=119….1
119÷16=7….7
7÷16=0….7
这样就计到487710（10）=7711E（16）
（3）浮点数：就是小数，浮点数的内存占用整数的两倍，所以js会把浮点数转化成整数。
（4）指数表示方法：1e1或者1e+1等表示1后面1个0，如果是1e+7，表示1后面7个0。同理如果是个减号，则是数字前面多少个0，3e-10，表示3前面有10个0.
（5）js中最小值保存在Number_MIN_VSLUE中，大多数浏览器中这个值是5e-324，最大值保存在Number_MAX_VALUE中，大多数中这个值是1e+308，如果超过这两个数，则是Infinity和-Infinity，判断一个数是不是有穷的用isFinite(),如果是则返回ture。
（6）NaN，是数字类型的非数字，不合任何人相等包括踏自己。任何与NaN操作都会返回NaN，0/0会得到NaN。isNaN()函数是讨厌数字的，如果是数字则返回false。
（7）Number(),可以把任何数据类型转换成数值，规则是，布尔值ture转换成1，false转换成0.数字字符串则转换成相应的数字，如果是空字符串则转换成0，如果包含十六进制则转换成十进制整数，其他的转换成NaN，null转换成0，undefined转换成NaN,如果是对象，先调用valueOf()方法，然后依照前面的转换规则，如果转换的是NaN，则调用toStirng()方法。
parseInt()只转换带数字的字符串，不转换true,false,函数,等。 从左到右一个一个转换，遇到不是数字就停止，但是可以认加减号，前面空格会忽略，在数字类型中要是 数字中间有空格会报错，但是在字符串中，遇到空格就截断停止转换。局部转换。可以第二个参数。
parseFloat()只转换带数字的字符串或者数字，认识一位小数点，比如’12.3.4’转换成12.3；parseFloat()只应用于解析十进制数字,所以只有一个参数。
3.4.5、String数据类型
用双引号或者单引号保卫。字符串有长度的，但是这个长度只能读取，不能够重新改写。把其他的数据类型改成字符串的方法，第一种是toString(),多数情况下不要括号里面不要参数，但是如果转换数值的toString()方法时，可以填写2或者8、16等进制数。
null和undefined是没有toString()的，所以有第二种方法，String(),能够把任何的数据类型转换成字符串，转换规则是：如果有toSting()，就用这个方法，如果是null和undefined，则返回”null”、”undefined”。
3.4.6、Object数据类型
创建对象的两种方式 var o={};或者 var o=new Object();
Object每个实例有下列属性和方法：constructor,保存用于创建当前对象函数。hasOwnProperty(),检查这个属性是否在当前实例中，括号里面的参数必须是字符串。

3.5、操作符
3.5.1、一元操作符（发生隐士转换）
1、递增和递减
分为前置和后置，前置的意思是位于操作变量前执行，后置表示的是操作变量后执行。比如
var n1=2,n2=20,n3=–n1+n2; //21 var n4=n1+n2 //21
var n1=2,n2=20,n3=n1– + n2; //22 var n4=n1+n2 //21
递增和递减能够把任何数据类型转换成数值。规则是如果是数字字符串则转换成相应的数字然后再加1，如果是非数字字符串，怎么返回NaN，如果是ture则变成1再加1.如果是fasle则是0然后加1。如果是对象，则先用valueOf()，转换成相应的值然后操作，如果转换的是NaN则用toString();
